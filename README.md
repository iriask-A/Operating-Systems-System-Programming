# Operating Systems and System Programming (C)

This repository contains my practice tasks and small projects related to **Operating Systems** and **System Programming** implemented in the **C programming language**.

The goal of this repository is to strengthen my understanding of low-level programming concepts, operating system mechanisms, and system-level development.

## Topics Covered

This repository may include implementations and experiments related to:

- Process creation and management
- Threads and concurrency
- Inter-process communication (IPC)
- Memory management
- File systems and file operations
- Signals and process control
- System calls
- Synchronization (mutexes, semaphores)
- Pipes and redirection
- Basic shell utilities

## Technologies

- C programming language
- GCC compiler
- Linux / Unix environment
- POSIX system calls
